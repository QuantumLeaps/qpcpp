About The Blinky-AVR Example
============================
This directory contains the Blinky example for the 8-bit AVRmega processor
family with the IAR-AVR compiler.

***
NOTE:
The Blinky example exceeds the code size limit of the free IAR-AVR KickStart
version and therefore it fails linking. Please use the unlimited IAR-AVR
toolset. 
***
